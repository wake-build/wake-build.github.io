# Options

```
usage: wake [-h] [-v] [-f CONFIG] [--dry-run] [-d DEFAULT_TAG] [-t TAG_PREFIX] [-p COSIGN_PROFILE] {build,pull,tag,push} ...

positional arguments:
  {build,pull,tag,push}

options:
  -h, --help            show this help message and exit
  -v, --verbose
  -f, --config CONFIG
  --dry-run
  -d, --default-tag DEFAULT_TAG
  -t, --tag-prefix TAG_PREFIX
  -p, --cosign-profile COSIGN_PROFILE
```
